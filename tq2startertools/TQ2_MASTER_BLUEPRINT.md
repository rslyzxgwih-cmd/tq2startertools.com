# Titan Quest 2 Starter Tools Master Blueprint

## 1. 产品定位

Titan Quest 2 Starter Tools 是一个面向英文玩家的游戏工具站。

目标用户：

* 新手玩家
* 回归 ARPG 玩家
* 想快速选择职业/流派的玩家
* 想知道开荒怎么玩的玩家
* 搜索 Titan Quest 2 build / class / skill / beginner guide 的玩家

核心定位：

不是新闻站。

不是 Wiki。

不是论坛。

而是：

Decision Tools for Titan Quest 2 Players.

帮助玩家快速回答：

* 我该选什么职业？
* 哪个流派适合新手？
* 哪些技能值得投入？
* 如何从开荒过渡到成型 Build？
* 不同玩法之间怎么选择？

## 2. 核心策略

借鉴 POE2 Starter Tools 的经验：

先做 Decision Pages。

不要一开始做大量内容页。

重点是：

* Beginner intent
* Build intent
* Class choice intent
* Skill choice intent
* Tool intent

## 3. MVP 核心页面

第一阶段只规划这些页面：

* /
* /builds
* /builds/beginner
* /classes
* /class-picker
* /skills
* /guides
* /guides/beginner
* /tools

暂不新增具体 build 页面。

暂不新增大量 skill 页面。

等官方资料明确后再扩展。

## 4. 未来内容集群

Guide Cluster：

* beginner guide

* class picker

## 5. 商业目标

短期：

* 被 Google 收录
* 获取 Titan Quest 2 早期搜索曝光
* 验证玩家搜索需求

中期：

* 建立 Topical Authority
* 推出 Class Picker / Build Advisor
* 形成稳定流量入口

长期：

* 复制 POE2 工具站经验
* 建立 ARPG 工具站矩阵
* 后续可承接广告、联盟、游戏工具产品

## 6. 不做什么

不追新闻热点。

不做论坛。

不抄 Wiki。

不在资料不确定时写大量具体数值。

不提前虚构未确认职业、技能、装备。
