# TQ2 MVP Scope

## MVP Goal

Launch a clean, crawlable, decision-focused Titan Quest 2 starter site before full game data is stable.

## Included in MVP

Pages:

* Homepage
* Builds Hub
* Beginner Builds
* Classes Hub
* Class Picker
* Skills Hub
* Beginner Guide
* Tools Hub

Features:

* Static content
* Decision cards
* Early access disclaimer
* Internal links
* Basic SEO metadata
* FAQ schema where appropriate
* Breadcrumb schema where appropriate

## Not Included in MVP

* Full build planner
* User accounts
* Database
* API
* Dynamic skill data
* Gear database
* Damage calculator
* Interactive map
* Forum
* Comments

## MVP Success Metrics

Google Search Console:

* Indexed pages
* Impressions for Titan Quest 2 queries
* First clicks
* Query diversity
* Page-level impressions

User behavior:

* Class Picker clicks
* Beginner Builds clicks
* Tool entry clicks

## MVP Launch Rule

Do not launch more than 8 core pages before search data appears.

Do not expand unless Google shows impressions or players show demand.
