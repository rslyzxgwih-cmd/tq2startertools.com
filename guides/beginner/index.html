Beginner guide: make the first build decision safer.

This guide explains the minimum context a new player needs before choosing a playstyle, mastery, build, or skill path.

Choose PlaystyleSee Beginner Builds

Beginner guide path

  1. 1. Choose what feels fun

    Start from melee, caster, ranged, pet, or hybrid.

  2. 2. Keep difficulty honest

    Choose easy or balanced until systems are familiar.

  3. 3. Follow one build path

    Do not scatter early skill and gear decisions.

Common beginner blockers

Should I start with endgame in mind?

Not in the Static MVP. Pick a path that teaches the game first, then optimize after systems are stable.

Should I pick hybrid early?

Only if complexity is the point. Most beginners should start with a clearer class direction.
